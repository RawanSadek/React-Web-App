### 1. Clone the repository
```bash
git clone https://github.com/RawanSadek/React-Web-App.git
cd React-Web-App
```

### 2. Install dependencies
```bash
npm install
# or
yarn install
```

### 3. Run the development server
```bash
npm run dev
# or
yarn dev
```

### 4. Build for production
```bash
npm run build
```

